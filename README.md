#Angular2 + Spring Boot Test Project

* 구성 : Angular-cli 1.0.0 + Spring Boot 1.5.1

* 실행
 1. frontend : frontend/src/main/frontend 로 이동 후, 아래와 같이 실행 (http://localhost:4200)
```
npm start
```

 2. backend : 최상위 디렉토리에서 아래와 같이 실행. (http://localhost:8080)
```
mvn clean install
cd backend
mvn spring-boot:run
```
